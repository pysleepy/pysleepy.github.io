Finally, I have deployed my GitHub Pages site -- with the help of AI (￣▽￣).

The first time I tried to build a personal website on GitHub was almost a decade ago. Between work, study, and life in general, the idea kept getting postponed again and again.

Each time I thought about returning to it, the web ecosystem seemed completely different from what I remembered. Back in the old days we built things with the classic LAMP stack (Linux–Apache–MySQL–PHP). Since then the ecosystem has evolved incredibly quickly, and at some point it felt overwhelming to catch up while also juggling everything else.

Thanks to the community and modern AI tools, I can finally set this up without spending weeks tweaking CSS, dealing with browser quirks (remember IE6? Oh my…), or worrying about infrastructure.

Well… almost.

I still managed to spend a few hours debugging why my blog posts had mysteriously disappeared -- only to discover that Jekyll hides posts dated in the “future”. The timezone mismatch between my machine and the server made my very first post look like it was written tomorrow. Classic.
(╯°□°）╯︵ ┻━┻

But in the end, everything works.

And now I can focus on what actually matters: writing.

So here it is -- a simple *Hello World*.

Only about a decade late.
